# How ❓

어떻게 이게 모던 Javascript 유틸리티 라이브러리?

# How how?

```
npm i How
```

or

```
yarn add How
```

# How Features

- array : 배열을 다루기 위한 함수들 제공
- function : 함수 호출을 다루는 함수를 제공
- number : 숫자를 다루는 함수를 제공
- object : 객체의 관한 함수를 제공
- string : 문자열에 관한 함수를 제공
- util : 날짜를 비롯한 유틸 함수를 제공
